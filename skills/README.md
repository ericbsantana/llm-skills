# Skills repository

Project-scoped Cursor skills. The agent uses these when working in this repo.

## Skills

| Skill | Description |
|-------|-------------|
| **mantine-ui** | Mantine React UI: setup, MantineProvider, theming, forms, optional packages. Use when using or adding Mantine. |

To add a skill: create a folder under `.cursor/skills/<skill-name>/` with a `SKILL.md` file (see [Cursor skills docs](https://docs.cursor.com/context/rules-for-ai#skills) or the create-skill guide).
