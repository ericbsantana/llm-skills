# Mantine reference

## Core packages

| Package | Purpose |
|--------|---------|
| `@mantine/core` | Components (inputs, buttons, overlays, layout, etc.) – required |
| `@mantine/hooks` | Hooks (state, UI, resize, etc.) – recommended |

## Optional packages (install only when needed)

| Package | Purpose | CSS import |
|--------|---------|------------|
| `@mantine/form` | Form state and validation | — |
| `@mantine/dates` | Date inputs, calendars | `@mantine/dates/styles.css` |
| `@mantine/charts` | Recharts-based charts | — |
| `@mantine/notifications` | Notifications system | `@mantine/notifications/styles.css` |
| `@mantine/code-highlight` | Code blocks with theme | `@mantine/code-highlight/styles.css` |
| `@mantine/tiptap` | Rich text editor | — |
| `@mantine/dropzone` | File dropzone | `@mantine/dropzone/styles.css` |
| `@mantine/carousel` | Embla carousel | — |
| `@mantine/spotlight` | Command palette | — |
| `@mantine/modals` | Modal manager | — |
| `@mantine/nprogress` | Navigation progress | — |
